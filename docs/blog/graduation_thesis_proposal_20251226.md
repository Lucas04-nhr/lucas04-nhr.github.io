尊敬的各位老师：

大家好！我是 <Plot trigger="click" effect="blur">`Name is redacted to protect privacy`</Plot>。今天，我很荣幸能在这里进行开题答辩，我的毕业设计课题是《大规模细菌基因组单核苷酸多态性分析揭示不同生境细菌进化关系》。校外指导老师是来自深圳华大生命科学研究院的邹远强研究员，校内指导老师是生命学院刘磊副教授。接下来，我将向大家汇报课题的背景、数据、流程和计划。请各位老师批评指正。

首先，谈谈研究背景。在细菌基因组学中，单核苷酸多态性（SNPs）是核心标记，用于解码基因组结构、描绘进化轨迹和构建高质量组装图。与水平基因转移（HGT）和结构变异（SV）相比，SNP 产生速率稳定（每代 10^-9^ 到 10^-8^ 个突变），适合作为分子钟估算最近共同祖先时间（tMRCA）和量化自然选择强度（如通过 $\pi$ 和 $\theta_W$ 比较）。然而，现有的进化研究多停留在基因组草图阶段，存在缺失位置信息、分辨率不足和对肠杆菌科关注不足等问题。我们的研究旨在通过基因组完成图结合全基因组 SNP 检测，构建高分辨率进化网络，揭示细菌在生态适应和抗生素抗性中的作用。

接下来，介绍数据来源。本数据库汇集了全球 26,323 个高质量细菌基因组（`CheckM2` 完整度平均 99.98%），覆盖 3,808 个独特物种，主要属于 <ruby>_Pseudomonadota_<rt>假单胞菌门</rt></ruby> 和 <ruby>_Bacteroidota_<rt>拟杆菌门</rt></ruby> 。数据地理分布广泛，亚洲占比最高（10,509 个），中国贡献最大（6,549 个）。基因组相关指标包括：contig 平均长度 313.45 bp，最大 contig 均值 4.38 Mb，总 contig 数量均值 2.28 个，GC 碱基含量均值 50%。生境来源多样，50% 来自人类（如肠道、血液），其余覆盖环境、动物和植物，支持微生物组学和病原体进化研究。

在研究流程方面，我们已基于参考基因组，使用 `parsnp` 软件处理数据，以物种为单位构建分析流程，生成 SNP 的 `.vcf` 文件，并初步构建进化树。未来计划包括：使用 `parsnp` 进行核心比对、SNP 矩阵和初始树构建，然后质量筛选和 SNP 过滤；重建高精度系统发育树，支持自助法和时间校准；进行群体遗传分析（结构、重组、选择），并整合 SNP-基因-功能注释；结合元数据构建时空传播网络，揭示进化路径、传播模式和选择驱动因素。

最后，以上是课题参考的部分文献，包括 Nature Communications 上的肠道细菌基因组景观研究，以及 Microbial Genomics 上的 BactSNP 管道等，这些为我们的方法提供了坚实基础。

## Q&A {#qa}

::: steps
1. 你的课题研究背景和意义是什么？为什么选择 SNP 作为主要分析标记？

    SNP 是细菌基因组中最常见的变异类型，产生速率稳定（每代 10^{-9}到 10^{-8} 个突变），适合作为分子钟估算 `tMRCA` 和量化自然选择（如通过 $\pi$ 和 $\theta_W$ 比较）。相比 HGT 和 SV，SNP 更可靠，用于构建高分辨率进化树。现有研究多基于草图基因组，分辨率不足，且对某些门类（如假单胞菌门）关注少。本课题利用 26,323 个高质量完成图基因组，结合全基因组 SNP 检测，揭示不同生境细菌的生态适应和抗生素抗性进化机制，具有重要的微生物组学和病原体进化研究意义。

2. 数据来源可靠吗？如何保证基因组的质量？

    数据汇集全球 26,323 个高质量细菌基因组，CheckM2 完整度平均 99.98%，覆盖 3,808 个独特物种，主要为假单胞菌门和拟杆菌门。地理分布广泛，中国贡献最大（6,549 个）。组装指标优秀：平均 contig 数 2.28，最大 contig 均值 4.38 Mb，GC 含量均值 50%。生境多样（50%人类相关），支持跨生态分析。这些数据来源于公共数据库（如 NCBI），已通过严格质量筛选，确保适合核心基因组比对和 SNP 调用。

3. 你提到使用 `parsnp` 软件，是什么工具？为什么选择它？

    `parsnp` 是 Harvest 套件中的核心基因组比对工具，专为大规模细菌基因组设计，能快速处理数千个基因组，输出核心 SNP 矩阵、VCF 文件和初步进化树。它基于 MUMs 锚定 LCBs，进行递归比对，支持重组过滤，适合我们的完成图数据和物种级分析。相比其他工具（如 `Snippy` 或 `Roary`），`parsnp` 无参考偏差，效率高，已在多篇高影响力论文中使用。

4. SNP 检测和过滤的具体流程是什么？如何避免假阳性？

    以物种为单位，选择合适参考基因组，使用 Parsnp 进行核心基因组比对，生成 SNP 的 VCF 文件。后续包括：质量筛选（覆盖度、质量分）、SNP 过滤（去除重组区，使用 `PhiPack` 等工具）、排除低频或聚类位点。未来还会整合上下文信息过滤。参考 `BactSNP` 等管道经验，确保高准确率，避免假阳性影响进化树构建。

5. 如何构建高精度系统发育树？会考虑重组影响吗？

    基于过滤后的核心 SNP 矩阵，使用 `RAxML` 或 `FastTree` 构建最大似然树，支持 Bootstrap 自助法验证分支可靠性。会进行时间校准（分子钟模型），并检测/过滤重组（`Parsnp` 内置 `PhiPack`）。对于细菌，重组常见，会额外使用 `ClonalFrameML` 等工具校正，确保树反映垂直遗传信号。

6. 群体遗传分析具体包括哪些指标？如何整合功能注释？

    包括群体结构（ADMIXTURE 或 PCA）、重组率估算（LD 衰减）、选择压力（$\frac{\pi}{\theta_W}$、$\frac{\mathrm{d}N}{\mathrm{d}S}$、Tajima's D、McDonald-Kreitman 测试）。SNP 会注释到基因/功能水平（使用 `Prokka` 或 `eggNOG`），识别选择驱动的位点，如抗生素抗性相关基因。最终整合元数据（生境、地理），揭示适应机制。

7. 时空传播网络如何构建？预期能揭示什么进化路径？

    结合进化树、元数据（采集时间、地点、生境）和 SNP 距离，使用 `Beast` 或 `TempEst` 进行祖先状态重建和传播路径推断。预期揭示细菌在不同生境（如人类肠道到环境）的传播模式、瓶颈事件和选择驱动因素，例如抗生素压力下的克隆扩张。

8. 你的课题创新点在哪里？与现有研究（如文献中提到的）有何区别？

    ==**创新在于**== 大规模高质量完成图（而非草图）结合全基因组 SNP，构建高分辨率进化网络；跨生境整合（人类、环境、动物），揭示生态适应；时空网络分析量化传播和选择。现有研究多聚焦单一物种或肠道（如 Nature Communications 论文），本课题覆盖多门类、全球数据，填补分辨率和生境多样性空白。

9. 课题可能遇到的困难和风险是什么？如何应对？

    - ==**困难包括**== 大规模数据计算量大（需高性能计算）；重组干扰进化信号；参考选择偏差。
    - ==**应对**== 使用 `Parsnp` 分区模式降低内存；多工具交叉验证（如 `Gubbins` 过滤重组）；物种内自动选参考（ANI/MUMi）。初步已处理部分数据，验证流程可行。

10. 你的研究计划和时间安排如何？毕业前能完成哪些部分？

    已完成数据收集和初步 `Parsnp` 处理。未来 6 个月：SNP 过滤、树构建和群体分析；接下来整合注释和元数据，构建时空网络；最后解读结果、撰写论文。预计毕业前完成核心分析和 1-2 篇论文投稿。

11. 如果某些物种核心基因组太小，导致 SNP 位点不足，怎么处理？

    会设置核心基因组阈值（如覆盖率>70%），对核心小物种单独分析或排除；或补充泛基因组方法（如 `Roary`），结合核心+辅助 SNP。但优先聚焦核心保守的物种，确保进化信号可靠。

12. 如何验证你的进化树和结论的准确性？

    通过 Bootstrap 支持率、与已知分类一致性、交叉工具验证（如 `IQ-TREE` vs `RAxML`）；模拟数据测试管道准确率；与文献树比较（如特定病原体进化）。
