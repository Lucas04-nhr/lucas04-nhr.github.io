## Introduction

StayUP Schedule is a lightweight class schedule app built with Flutter.

<RepoCard repo="Shiroko114514/StayUP-Calendar" />

The project started from a simple goal: keep the schedule experience clean and focused, without unnecessary complexity in daily use. During development, AI tools were also used for learning and implementation, so this is a practical project that was built and improved iteratively.

## Supported Platforms

This is a standard Flutter multi-platform project that supports the following platforms:

- Android
- iOS
- macOS
- Windows
- Linux
- Web

::: note

Besides the Android and iOS versions, other versions are currently in the early stages of development and may not be fully functional. The Android version is the main focus for now, and other platforms will be improved over time.

:::
