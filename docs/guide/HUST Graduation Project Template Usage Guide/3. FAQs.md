## 代码块无法正确渲染高亮 {#code-block-highlight}

请确保已经正确安装了 `pygments`，并且在 `.latexmkrc` 文件中正确配置了环境变量路径。

## 行间公式无法正确编号

请确保使用的是 `equation` 环境来输入行间公式，而不是使用 `$$ ... $$` 语法。例如：

```tex
\begin{equation}
  E = mc^2
\end{equation}
```

## 图片无法正确显示 {#image-display}

请确保图片文件路径正确，并且图片格式被支持（如 PNG、JPEG、PDF 等），推荐使用 PDF 格式以获得最佳效果。

## 参考文献无法正确生成 {#bibliography-issue}

确保已安装 BibTeX，并在编译时正确执行参考文献相关的步骤（使用 `latexmk` 可自动完成）。如遇 Windows 平台上 biber 或相关工具缺失的问题，请参阅 [Rapheal](https://github.com/chide-org) 提供的[Windows 简明教程（含 biber 问题修复）](/docs/hust-gp-template/wsl-biber-fix/) 获取针对性的解决方法。

## 字体显示异常 {#font-issue}

请确保已经正确安装了所需的字体，并且在编译时使用了 XeLaTeX 引擎。

## 无法通过编译 {#compilation-issue}

请仔细检查编译日志，查找具体的错误信息，并根据提示进行修正。关于缺少宏包的解决办法可参考[这里 (English only)](https://yihui.org/tinytex/#maintenance)。

<LinkCard icon="ic:round-help-center" title="维护说明（英文）" href="https://yihui.org/tinytex/#maintenance" description="点击查看 TinyTeX 官方维护说明" />

## 如何使用“参考文献译文本”或“开题报告”模板？ {#template-usage}

将 `misc/translation` 或者 `misc/proposal` 目录中的相关文件移动并覆盖根目录下的同名文件即可。请注意，“开题报告”相关模板仍在完善中，相关内容将在**不晚于第二学期开学前**进行补充。

## 如何处理“参考文献译文本”中的参考文献条目？ {#translation-bibliography}

根据学校要求，“参考文献译文本”需要将原始参考文献的参考文献条目一并插入文中，推荐采用 Scopus 进行批量导出 `.bib` 文件并通过正则表达式删去 `note` 字段，相关替换的正则表达式如下，以 VS Code 为例：

```
^\s*note\s*=\s*\{[^}]*\}\s*,?\r?\n?
```
